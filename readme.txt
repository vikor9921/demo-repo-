15 примеров команд системы контроля версий Git:
1) $ git add - добавление изменений. Применяется, когда Вы готовы зафиксировать изменения и создать новый коммит.
2) git status - проверка текущего состояния репозитория и списка изменений. Применяется для того, чтобы узнать какие файлы были изменены, добавлены или удалены.
3) $ git reset - используется для отмены изменений. Используется, если Вы хотите удалить изменения, которые ещё не были зафиксированы в коммите.
4) $ git commit -m "" - создание нового коммита с зафиксированными изменениями. Применяется после завершения работы над изменениями и когда Вы готовы добавить их в историю репозитория.
5) $ git init - инициализация нового репозитория. Применяется, когда вы хотите создать новый проект.
6) $ git push - отправка локальных коммитов на удалённый репозиторий. Применяется, когда вы готовы поделиться вашими изменениями с другими участниками.
7) $ git checkout - переключение между ветками. Применяется, когда Вы хотите начать работу в определённой веткепроекта.
8) $ git branch - просмотр списка веток в репозитории или создание новой ветки. Применяется, если Вы хотите работать в определённой ветке.
9) $ git diff HEAD - используется для просмотра различий между последним коммитом и текущим состоянием в рабочем каталоге.
10) $ git pull - получение последних изменений с удалённого репозитория и объединение их с локальной веткой. Применяется для обновления своей локальной версии проекта до актуального состояния.
11) $ git merge (название ветки) - объединение изменений из одной ветки в другую. Используется для слияния веток.
12) $ git diff --staged - используется для просмотра различий между последним коммитом и текущим сеансом. Полезно, когда мы хотим посмотреть, какие изменения были добавлены, но ещё не были зафиксированы в коммите.проекта до актуального состояния.
13) $ git log - просмотр истории коммитов в репозитории. Применяется, когда вы хотите просмотреть список произошедших изменений.
14) git checkout -- (название файла) - отмена изменений в файле. Используется, если Вы хотите удалить изменения в файле.
15) $ rm -r - удаление каталога со всеми входящими в него файлами.
